# MediSphere

MediSphere is a simple Hospital Management System built with Python, Tkinter, and Object-Oriented Programming.

## Features

- Add patients
- View doctors
- Book appointments
- Generate and download bills
- Handle emergency requests
- Includes sample patient and doctor data

## Technologies Used

- Python
- Tkinter
- OOP Concepts
