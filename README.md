# MediSphere - Hospital Management System

MediSphere is a Python Object-Oriented Programming project built using Tkinter. It is a desktop application for managing patients, doctors, appointments, billing, and emergency services in a hospital.

## Features

- Add patient details
- View available doctors
- Book appointments
- Select doctors based on illness
- Select available appointment slots
- Generate hospital bills
- Download bills as text files
- Calculate appointment fee, doctor fee, room charge, and service charge
- Handle emergency requests
- Ambulance option for emergency cases
- Sample patient and doctor data included
